Αυτή η σελίδα δείχνει πως κανουμε ajax με jquery. Αν δεν έχουμε σηκώσει τα services της java τότε θα δούμε ότι το ajax αποτυγχάνει(console του chrome). H σελίδα που περιέχει το api της βιβλιοθήκης είναι http://api.jquery.com/jquery.ajax/ Τα πιο σημαντικά options είναι : url: Το url ου θα κληθεί. data: Τα data που θα στείλω στον server. Αν είναι GET θα προστεθούν σαν query params. dataType: Ποια θα είναι η μορφή της απάντησης που θα δώσει ο server. Μπορεί να έχει διάφορες μορφές, εμείς κυρίως θα δεχόμαστε json. contentType: Η μορφή με την οποία στέλνουμε τα data στον server. Σχεδόν πάντα θα τα στέλνουμε σαν json, δηλαδή η παράμετρος αυτή θα έχει τιμή application/json. Στο παράδειγμα μην σηκώσεις αρχικά το service της java. Πάτα το button και δες στο network των development tools του chrome τι κάνει. Θα δεις ότι έχει μια εγγραφή με all, η οποία έχει status failed
